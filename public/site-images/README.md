Put your images in this folder with these exact filenames:

1. `01-hero.jpg`
   Used for the home hero background

2. `02-about.jpg`
   Used for the About section image

3. `03-story.jpg`
   Used for the Story section image

4. `05-gallery-1.jpg`
   Used for Gallery image 1

5. `06-gallery-2.jpg`
   Used for Gallery image 2

6. `07-gallery-3.jpg`
   Used for Gallery image 3

7. `08-gallery-4.jpg`
   Used for Gallery image 4

8. `09-gallery-5.jpg`
   Used for Gallery image 5

9. `10-gallery-6.jpg`
   Used for Gallery image 6

Notes:
- The trailer thumbnail is still using the existing placeholder image for now.
- You can use `.jpg`, `.png`, or `.webp`, but if you change the extension you must also update the paths in `src/site-content.ts` and `data/site-content.json`.
- Right now the app is already wired to these filenames.
- If an image does not appear, it usually means the filename does not match exactly.
